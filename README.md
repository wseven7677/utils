# w7677utils
> some utils of mine
> 集成了自己常用的工具函数.

### 安装方法
- 安装：
```
npm install --save w7677utils
```
- 引入：
```
var utils = require('w7677utils');
// or
import utils from 'w7677utils';
```
- 使用：
```
utils.xxxxxx()
```
